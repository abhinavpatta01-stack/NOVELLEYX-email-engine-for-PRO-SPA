1. Brand Profiles & Presets

The Brand Settings panel allows you to configure global brand parameters and easily switch between different company presets. The active brand profile affects the email header styling, links, footers, logo images, and target fallback actions.

Key Fields & Custom Presets

  • Select Active Profile: Quickly apply preset styles from saved companies. By default, the system provides a core Novelleyx profile. Newly created profiles populate here instantly.
  • Add Company Profile: Opens a modal dialog to define a new brand archetype. You can specify a company name, a main fallback website URL, a primary theme hex color, and a brand logo URL. The system will auto-generate preset templates for this company.
  • Brand Name: Controls the textual representation of your brand in the email header and footer metadata.
  • Fallback Link URL: The safety URL configuration. If a customer clicks on your promotional banner graphic outside of your specifically mapped hotspot click zones, they will be automatically redirected to this URL.
  • Inboxes Preheader: The invisible preview text that appears next to the subject line in modern email clients (like Gmail or Outlook). Crafting a high-conversion preheader significantly boosts email Open Rates.

Pro-Tip: Always include an emoji in your Preheader text (e.g. ⚡, 🎁, 🔥) to increase visual standing in customer lists.

2. Promotional Banner Graphics

The promotional banner is the visual centerpiece of your email campaign. The Novelleyx editor provides two distinct loading options tailored to your workflow requirements.

Image Loading Modes

  1. Local Upload (Base64 Mode): Enables direct local image selection from your machine. The engine reads the file, converts it into a Base64 data string, and stores it in-memory.

    Limitation: Base64 encoding inflates the output HTML size. Useful for offline drafting or quick deployments.

  2. External CDN URL Mode: Highly recommended for production email lists. You paste the absolute web URL (e.g., https://mycdn.com/assets/campaign-banner.png) where your image is hosted.

    Advantage: Generates extremely lightweight HTML code blocks, avoiding recipient mailbox clipping.

Caution: The image file size limit for local uploads is restricted to 3MB. If your graphic exceeds this, compress it first or host it on an external CDN server.

3. Hotspot Area Editor

The Hotspot Canvas is the interactive engine. It allows you to draw multiple non-overlapping click areas over your promotional banner graphic, creating an image-map with distinct redirects.

Drawing & Modifying Hotspots

  • Drawing a Hotspot: Click and drag anywhere on the visual banner image canvas. A translucent blue overlay rectangle will expand. Release the mouse button to complete.
  • Mapping: On completion, a hotspot config dialog appears. Assign a clear description label (for accessibility and testing) and the redirection link URL.
  • Interactive Controls:
    • Select: Click inside any hotspot boundary to highlight it.
    • Resize: Select a hotspot, click and hold the bottom-right grabber handle, and drag to resize.
    • Reposition: Select a hotspot, click inside its bounds (avoiding the delete button or resize handle), and drag it to any other region on the canvas.
    • Delete: Click the small red cross (×) in the top-right corner of a selected zone to remove it.

User Tip: You can edit a hotspot's redirect link at any time by clicking "Edit Link" in the hotspot list below the canvas or by double-clicking the hotspot rectangle overlay.

4. Secondary CTA Block

To ensure high compliance across old email clients that might block images, the engine provides an optional secondary CSS-styled Text Call-To-Action (CTA) block beneath the banner graphic.

Configuration Parameters

  • Headline: High-impact title formatted in a large bold font (e.g. "Claim Your Early Access Now").
  • Body Description: Rich HTML-friendly copywriting details describing the promotional offer or instructions.
  • Button Text & Link: Customizable button labels and distinct redirect URLs.
  • Color Sync: Fine-tune the CTA button background color and the text color using the hex input fields or the interactive visual color picker.

5. Footer & Social Configurations

The email footer contains corporate ownership lines, physical address fields, unsubscribe compliance, and social media channels. You can toggle this section on or off based on list requirements.

Social Channels

Activate individual checkboxes for major platforms (Facebook, X, Instagram, LinkedIn, YouTube) and input the absolute link to your brand page. The compiled HTML will output beautiful responsive SVG social icons matched to your brand style configuration.

6. Email Transmission System

Once your template is finalized, you can test or dispatch the campaign directly from the interface using the "Send Email" action in the Preview Tabs. The system provides two live delivery paths.

Method A: SMTP Server Relay

Connects directly to your own SMTP server (e.g., SendGrid, Mailgun, Amazon SES, or Gmail) via the secure public SMTPJS client-side relay.

POST https://smtpjs.com/v3/smtpjs.aspx? Headers: Content-Type: application/x-www-form-urlencoded Payload: { Host, Port, Username, Password, Secure, To, From, Subject, Body, Attachments }

Setting Up SMTP Credentials:

  • SMTP Host & Port: Provide your server endpoint. Standard secure ports are 587 (STARTTLS) or 465 (SSL).
  • SMTP Credentials: Input your SMTP username and authentication password.
  • Attachments: Any local files loaded into the dropzone (max 3MB per attachment) will be parsed, encoded to base64, and attached to the raw SMTP mime structure.

Important SMTP Tip: If using a Gmail account, you cannot use your regular account password due to security settings. You must log into Google Account settings, enable Two-Factor Authentication, and generate an App Password specifically for SMTPJS.

Method B: API Webhook Dispatch

Routes the compiled HTML code, sender metadata, and base64 attachment buffer objects as a JSON payload to a custom webhook URL (e.g., Make.com, Zapier, or a custom REST API endpoint).

POST [Your Webhook URL] Headers: Content-Type: application/json Payload: { "from": "sender@company.com", "to": "recipient@company.com", "subject": "Email Subject", "html": "<html>...</html>", "attachments": [ { "filename": "doc.pdf", "contentType": "application/pdf", "content": "BASE64..." } ] }

7. Novelleyx AI Assistant

The Novelleyx AI Assistant (found in settings section 6) is an automation helper built directly into the sidebar panel. It uses a client-side NLP intent-parsing model to instantly configure theme styles, copywriting, navigation link structures, and preset configurations based on simple language instructions.

How to Operate the AI Assistant

  1. Select your Automation Focus Scope (Full Auto brand generation, Visual style only, or Promotional CTA copywriter only).
  2. Write a plain English command in the prompt input field (e.g. "Create an organic wellness brand styled around botanical skincare" or "A sales campaign for cyberpunk neon streetwear").
  3. Click Generate Brand Experience.
  4. Observe the AI Reasoning Core Terminal console as the engine parses archetypes, maps variables, outputs step-by-step thinking logs, and applies parameters directly to the editor canvas in real-time.

8. PDF Export & Print Tips

This user manual is formatted dynamically for both digital reading and physical print layouts. Follow these steps to generate a professional PDF copy for offline reference.

How to Save this Manual as a PDF:

  1. Click the floating Save as PDF / Print button at the bottom-right corner of this page (or press Ctrl + P on Windows / Cmd + P on macOS).
  2. In the printer dialog prompt that appears, change the Destination dropdown select menu to Save as PDF.
  3. Ensure the Margins parameter is set to Default or Custom with clean spacing.
  4. Check the checkbox for Background Graphics to preserve visual indicators, icons, and warning box highlight colors.
  5. Click Save and select your destination directory.
